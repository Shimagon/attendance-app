## 🎯 3ステップセットアップ

### ステップ1️⃣: アイコン生成（30秒）

```bash
open generate-icons.html
```

1. ブラウザで自動的にアイコン表示
2. 2つの「ダウンロード」ボタンをクリック
3. ダウンロードしたファイルを移動:

```bash
mv ~/Downloads/icon-*.png icons/
```

### ステップ2️⃣: サーバー起動（10秒）

```bash
./start-server.sh
```

表示されるiPhone用URL（例: `http://192.168.1.157:8000`）をメモ

### ステップ3️⃣: iPhoneでアクセス（1分）

1. **iPhoneのSafariで** メモしたURLを開く
2. 名前を入力して「始める」
3. **現在時刻が秒単位で更新されることを確認** ✨
4. 出勤ボタンをタップして動作確認
5. Safari下部の **共有ボタン** > **ホーム画面に追加**

**完了！** 🎉

## 📊 スプレッドシート連携（オプション）

データをGoogleスプレッドシートに記録するには：

**詳細手順**: [GOOGLE_APPS_SCRIPT_SETUP.md](GOOGLE_APPS_SCRIPT_SETUP.md)

### 簡易版手順:

1. **スプレッドシートを開く**:
   ```
   https://docs.google.com/spreadsheets/d/1MaCsDpwYOtNn8hqywq300hThzMIXOhPnkS6LvcUskuk/edit
   ```

2. **3つのシートを作成**:
   - `研修生マスタ` - ヘッダー: 研修生ID, 氏名, ステータス
   - `打刻記録` - ヘッダー: 日付, 研修生ID, 氏名, 出勤時刻, 退勤時刻, 勤務時間
   - `課題完了記録` - ヘッダー: 完了日時, 研修生ID, 氏名, アプリURL, 判定

3. **Apps Scriptを設定**:
   - 拡張機能 > Apps Script
   - `google-apps-script.js` の内容をコピー&ペースト
   - デプロイ > 新しいデプロイ > ウェブアプリ
   - デプロイURLをコピー

4. **アプリに設定** (既に設定済み):
   ```javascript
   // js/app.js の API_CONFIG
   APPS_SCRIPT_URL: 'あなたのデプロイURL'
   ```

## 🎨 新機能: 現在時刻表示

出勤ボタンの上に **現在時刻が秒単位** で表示されます：

```
┌─────────────────────────┐
│      現在時刻           │
│    21:30:45            │
│ 2025年11月29日（金）    │
└─────────────────────────┘
```

- 🕐 **HH:MM:SS** 形式で表示
- 🔄 **1秒ごと**に自動更新
- 📅 曜日も表示
- 💜 美しいグラデーション背景

これで **少し待ってから退勤** などの操作がしやすくなりました！

## 📱 使い方

### 出勤
1. **現在時刻を確認**
2. 出勤ボタンをタップ
3. 勤務時間がリアルタイム表示開始

### 退勤
1. **現在時刻を確認**（数秒待つことも可能）
2. 退勤ボタンをタップ
3. 勤務時間が自動計算される

### 課題完了報告
1. 課題完了ボタンをタップ
2. 確認ダイアログで「OK」
3. スプレッドシートに記録される

## 🆘 困ったら

### iPhoneでアクセスできない
```bash
# IPアドレスを再確認
ipconfig getifaddr en0
```
新しいIPで再度アクセス

### 時刻が表示されない
ブラウザのコンソールでエラーを確認：
- Safari > 開発 > Webインスペクタ

### スプシに記録されない
1. Apps ScriptのデプロイURLを確認
2. `js/app.js` の `APPS_SCRIPT_URL` を確認
3. ブラウザコンソールでエラーログを確認
